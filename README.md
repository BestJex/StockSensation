# StockSensation
The sensation data of stock </br>
这是一个股市舆情分析系统，通过输了股票代码除了能够提供基本的个股历史交易行情，还能展示基于情感字典和机器学习的舆情分析预测</br>
目前的功能：

* 个股历史交易行情
* 个股相关词云展示
* 情感字典舆情预测
* 朴素贝叶斯舆情预测
